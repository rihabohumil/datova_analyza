# datova_analyza
# Sales Data Analysis Project

Python project focused on sales data analysis using pandas and OOP principles.

## Features
- CSV data loading
- Revenue calculation
- Product analysis
- Region analysis
- Excel export
- Data visualization with matplotlib

## Technologies
- Python
- pandas
- matplotlib
- OOP

## Run project

```bash
pip install -r requirements.txt
python Datova_analyza_OOP.py
```
